# Sales Analysis Dashboard (Power BI)

## Overview

This project analyzes sales data (₹2.26M) across regions, categories, and customers using Power BI.

## Key Insights

* West region has the highest sales
* Technology category dominates revenue
* Sales peak in November–December (seasonality)
* Overall growth observed from 2016 to 2018

## Tools Used

* Power BI
* Excel

## Features

* Interactive dashboard with slicers
* KPI tracking
* Trend analysis
* Customer segmentation
